Impact: duplicated behavior for a core flow primitive increases drift risk and testing cost.

Impact: boilerplate-heavy control-flow; harder to apply cross-cutting changes uniformly.

Impact: likely future divergence in cache behavior and error handling.

Impact: redundant code and repeated full cloning can become expensive as data volume grows.

Impact: potential runtime failure on cyclic metadata objects.
